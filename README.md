# AtenKey

AtenKey is a desktop shortcut controller for Home Assistant. It lets you trigger lights, scenes, automations, scripts, and other Home Assistant actions from global keyboard shortcuts on your computer.

## Highlights

- Global keyboard shortcuts for Home Assistant actions
- Multiple Home Assistant connections
- Secure token storage with the operating system keychain
- Import and export for app settings
- Signed and notarized macOS builds
- Automatic updates through Tauri's secure updater

## Download

Download the latest public release from:

https://github.com/xuonghuynh/AtenKeyPublic/releases/latest

## Updates

AtenKey checks this repository for update metadata:

https://github.com/xuonghuynh/AtenKeyPublic/releases/latest/download/latest.json

Update artifacts are signed and verified by the app before installation.

## Security

Home Assistant access tokens are stored locally using the system keychain. macOS builds are signed and notarized with Apple Developer ID before distribution.

## Repository Purpose

This public repository is used for AtenKey release distribution and update metadata. Source code, issue tracking, and development history may be managed separately.
